Control Server

Constellation Control Server helps you configure and control multiple interactives from anywhere in your museum.

Permissions

Permissions enable you to adjust how much Control Server can affect the system.

Smart Restart

Smart Restart helps you compenstate for a PC that occasionally drops a Wi-Fi connection. If the app cannot connect to Control Server for a given period of time, it can initiate a restat of the PC to regain the Wi-Fi connection.

Remote display

Advanced For more complex setups, it's possible to run Constellation Apps on this PC, but use a different device as the display. The most common use case for this approach is using an iPad or other tablet as the display.

This change will not take effect until you restart Constellation Apps.
