Exhibitera Apps
Welcome to Exhibitera Apps!

Exhibitera Apps is a collection of software that helps you put your digital content front-and-center. It's a powerful yet intuitive way to build guest-facing digital interactives for use on the museum floor.

Since this is your first time running Apps in this directory, we need to set up a few things before you can get started.

Are you using Hub?

Exhibitera Hub helps you configure and control multiple interactives from anywhere in your museum. With Hub, you can:

  • See the status of every interactive using Apps.
  • Power on or off many types of projectors.
  • Create daily schedules that automatically power on or off devices, change digital signage, or even switch interactives.
  • Collect and log evaluation data and analytics.
  • Track exhibit maintenance.

You must specifiy Control Server's IP address and port. If you don't know these values, contact your system administrator.
Will the app run entirely on this PC?

For more complex setups, it's possible to run Exhibitera Apps on this PC, but use a different device as the display. The most common use case for this approach is using an iPad or other tablet as the display. A couple caveats:

  • Depending on the app you are using and the quality of your network, performance may be degraded on a secondary device.
  • You should use Guided Access on iPadOS or a kiosk app on Android to lock down the device (Exhibitera can't do this for you).
  • You cannot access the same instance of Apps on multiple devices at once.

What should we call this component?

Since we're using Hub, we need to identify this component. Each app instance needs:

An ID, which uniquely identifies this component. A good ID might be something like Sports Intro Video.

A group, which collects together related components. You might choose to group by gallery, such as Sports Gallery, by purpose, like Video, or something else.

You must enter both an ID and a group for this component.
